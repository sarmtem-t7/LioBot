# Техническое задание — LioBot

## Описание проекта

**LioBot** — Telegram-бот для христианского книжного сообщества.  
Помогает пользователям находить подходящие христианские книги, отвечает на вопросы и каждое утро присылает мотивационное сообщение.

---

## Технологический стек

- **Язык:** C# (.NET 8)
- **Telegram:** Telegram.Bot
- **AI:** Anthropic Claude API (claude-sonnet-4-6 или claude-haiku-4-5)
- **БД:** SQLite + Microsoft.Data.Sqlite
- **Расписание:** Quartz.NET

## Функциональные требования

### 1. Подбор книг
- Пользователь описывает свою ситуацию или запрос
- Claude анализирует запрос с учётом тегов и описаний книг из БД
- Бот возвращает рекомендации с кратким объяснением выбора

### 2. Утреннее сообщение
- Каждый день в `ScheduleTime` (по умолчанию 08:00 MSK) всем зарегистрированным пользователям отправляется уникальное утреннее сообщение
- Сообщения не должны повторяться — Claude генерирует их динамически
- Содержание: библейский стих, краткое наставление или вдохновение

### 3. Обработка сообщений
- Бот корректно обрабатывает **любые** сообщения, не только команды `/start`, `/help`
- Свободный диалог через Claude с сохранением контекста пользователя

---

## Требования к стилю общения

- Все ответы — **на русском языке**
- Тон — **тёплый, дружелюбный, христианский**
- Без формализма, как разговор с близким другом в вере

## База данных

### Таблица `Users`
| Поле | Тип | Описание |
|------|-----|----------|
| Id | INTEGER PK | Внутренний ID |
| TelegramId | BIGINT | ID пользователя Telegram |
| Username | TEXT | Username в Telegram |
| FirstName | TEXT | Имя |
| RegisteredAt | DATETIME | Дата регистрации |

### Таблица `Books`
| Поле | Тип | Описание |
|------|-----|----------|
| Id | INTEGER PK | ID книги |
| Title | TEXT | Название |
| Author | TEXT | Автор |
| Description | TEXT | Описание книги |
| Tags | TEXT | Теги через запятую |

## Заметки по реализации

- `ClaudeService` — обёртка над Anthropic API, принимает системный промпт и сообщение пользователя
- `BookService` — загружает книги из БД, форматирует их для передачи в промпт Claude
- `SchedulerService` — Quartz.NET job, триггерится по `ScheduleTime`, рассылает утренние сообщения
- `MessageHandler` — роутинг входящих сообщений Telegram, вызывает нужные сервисы
- `DatabaseContext` — инициализация SQLite, миграции при старте
